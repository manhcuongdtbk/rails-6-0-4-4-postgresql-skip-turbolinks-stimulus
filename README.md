# RAILS 6.0.4.4

## How it is created

`rails _6.0.4.4_ new rails-6-0-4-4-postgresql-skip-turbolinks-stimulus --database=postgresql --skip-turbolinks --webpack=stimulus`

## What is included

- ruby 2.6.9
- basic rails 6.0.4.4
- postgresql database
- skip turbolinks
- webpacker 4.3.0 (installed by `rails new`) with stimulus 3.0.1
